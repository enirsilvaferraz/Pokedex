# Política de Privacidade — Pokedex

**Última atualização:** 9 de março de 2025

## 1. Introdução

Esta Política de Privacidade descreve como o aplicativo **Pokedex** (“App”) lida com informações no seu dispositivo. O App é oferecido para uso “como está” e não coleta, armazena nem compartilha dados pessoais dos usuários.

## 2. Dados que NÃO coletamos

O Pokedex **não coleta** dados pessoais, tais como:

- Nome, e-mail ou endereço
- Dados de localização
- Identificadores de dispositivo ou publicidade
- Histórico de uso ou preferências vinculadas à sua identidade

Não há cadastro, login nem qualquer forma de identificação de usuário.

## 3. Dados armazenados localmente no seu dispositivo

Para melhorar a experiência e permitir uso offline, o App **armazena localmente** no seu aparelho as respostas da **API pública PokeAPI** (https://pokeapi.co). Esses dados incluem apenas informações públicas sobre Pokémon (nomes, imagens, estatísticas, descrições, etc.).

- **Onde ficam:** apenas no armazenamento interno do seu dispositivo (cache/banco de dados local).
- **Quem acessa:** somente o próprio App no seu aparelho. Nenhum servidor nosso recebe ou processa esses dados.
- **Finalidade:** exibir conteúdo da Pokedex e permitir uso sem internet após o primeiro carregamento.

Esses dados **não são enviados** para os nossos servidores nem para terceiros, pois não operamos servidores que recebam dados do App.

## 4. Uso da internet

O App solicita permissão de **acesso à internet** apenas para:

- Buscar dados da API pública PokeAPI (https://pokeapi.co).
- Não usamos a conexão para enviar dados pessoais ou de uso para terceiros.

Consulte os termos e a política de privacidade da PokeAPI para informações sobre o uso dos dados por esse serviço: https://pokeapi.co.

## 5. Compartilhamento de dados

O Pokedex **não vende, não aluga nem compartilha** dados pessoais com terceiros, pois não coletamos dados pessoais. Os dados da API que ficam no seu dispositivo permanecem apenas no seu aparelho.

## 6. Segurança

Os dados armazenados localmente ficam no armazenamento do aplicativo no seu dispositivo, sujeitos às proteções do sistema operacional (Android). Recomendamos manter o aparelho atualizado e protegido por senha ou bloqueio.

## 7. Crianças

O App não coleta dados pessoais. Se o aplicativo for usado por menores, não há coleta de informações que identifiquem a criança.

## 8. Alterações nesta política

Podemos atualizar esta Política de Privacidade. A data da “Última atualização” no topo da página será alterada quando houver mudanças. O uso continuado do App após alterações constitui aceitação da política atualizada.
